An nth Partial Sum:

$$ \sum_{i=1}^n i = \frac{n(n-1)}{2} $$

A code example:

int main(int argc, char *argv[]) 
{
	int count = 0;
	int max = argv[1];

	for (int i = 1; i <= max; i++) 
	{
		count += i;
	}
	
	std::cout << "Sum: " << count << std::endl;
}
